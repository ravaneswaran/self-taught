To pull and install couchbase image
===================================
$ sudo docker pull couchbase

To run the couchbase docker container
=====================================
$ sudo docker run -d --name db -p 8091-8097:8091-8097 -p 9123:9123 -p 11207:11207 -p 11210:11210 -p 11280:11280 -p 18091-18097:18091-18097 couchbase

To get the container id
=======================
$ sudo docker ps

To stop the container
=====================
$ sudo docker stop <container-id>
