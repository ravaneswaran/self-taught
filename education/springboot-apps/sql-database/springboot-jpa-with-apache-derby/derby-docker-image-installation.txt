To pull and install derby from docker hub
=========================================
$ sudo docker pull az82/docker-derby

To run the mysql container
==========================
$ sudo docker run --name spring-boot-derby --hostname spring-boot-derby -p 1527:1527 -d az82/docker-derby:latest

To identify the container id of mysql
=====================================
$ sudo docker ps

To stop the container
=====================
$ sudo docker stop spring-boot-derby
