To pull the docker image from the docker hub
============================================
$ sudo docker pull apache/activemq-classic

To run the local instance of the activemq docker container
==========================================================
$ sudo docker run -d -p 61616:61616 -p 8161:8161 apache/activemq-classic

To get the container id
=======================
$ sudo docker ps

To stop the container
=====================
$ sudo docker stop <container-id>
